Regression

Regression analysis (or, in short, regression) is the task of estimating a model that can predict the value of a (dependent) variable given the values of other (independent) variables usually called features. For example, if we know the size and the number of bathrooms of a house in a neighbourhood, we might be able to predict its price if we have these data for a few other houses in the same neighbourhood. There are many algorithms to create regression models; as we will see, Hivemall offers a very convenient "general" regressor (along with other ones) that can be easily customized and used for prediction.

scala> val trained = spark.sql("SELECT feature, avg(weight) as weight FROM (SELECT train_regressor(features, target, '-loss squared -opt AdaGrad -reg No -iters 20') AS (feature, weight) FROM e2006_train_table) t GROUP BY feature")

:::scala
scala> val pred = spark.sql("SELECT t.rowid, sum(m.weight * t.value) as predicted FROM e2006_test_table_exploded t LEFT OUTER JOIN regr_model m USING (feature) GROUP BY t.rowid")
scala> pred.createOrReplaceTempView("pred")

:::scala
scala> val eval = spark.sql("SELECT rmse(predicted, actual) as RMSE, mse(predicted, actual) as MSE, mae(predicted, actual) as MAE, r2(predicted, actual) as R2 FROM (SELECT t.target actual, p.predicted predicted FROM e2006_test_table t JOIN pred p USING (rowid))")
